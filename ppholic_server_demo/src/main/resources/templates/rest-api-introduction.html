RESTful API Introduction


Hello World,


Besides the website directly implemented on the server, the RESTful API is designed and the client application based on the RESTful API is implemented.

As a demo version of the client application of the travel agency website, it's implemented with following features:

1. JWT Authentication: Stateless communication with server by JWT Authentication.

2. Full Member Function: Identical with the website implemented on the server.

(1) Product Function: Browsing, searching, sorting, management.

(2) External Client Utility: Registration, information update, dedicated shopping cart edition and payment, shopping record.
